RoyalCommands Documentation

Up to date (5.0.0).

Welcome to the documentation for RoyalCommands. The Commands page houses every command in RoyalCommands and is automatically updated when changes are made to the plugin. Formats contains the various formats that are accepted as arguments for commands. Finally, Signs has the signs that can be created with RoyalCommands (but it is currently not completed).

This site is incomplete and subject to change. Thank you for understanding! If you would like to help contribute to this site, please visit the GitHub.
